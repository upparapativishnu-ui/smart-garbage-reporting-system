# Smart Garbage Reporting System

A smart waste management and garbage reporting web application built using Flask, HTML, CSS, JavaScript, and SQLite.

## Features

- User garbage reporting system
- Admin dashboard
- Worker dashboard
- Complaint management
- Image upload support
- Real-time garbage issue tracking
- Clean and responsive UI

## Technologies Used

- Python Flask
- HTML5
- CSS3
- JavaScript
- SQLite
